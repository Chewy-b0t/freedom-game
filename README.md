| Key | Action |

| **Q** | 🔔 Wake Up — convert nearby enemies for bonus sats |

## ⚔️ Gameplay

### Core Loop

```
Fight Enemies → Collect Sats → Buy Upgrades → Survive Waves → Beat Bosses
      ↓                                              ↑
   Convert ──→ Recruit Allies ──→ Power-Ups ──────────┘
```

1. **Fight** through waves of war mongers, contractors, and central bankers
2. **Collect sats** (satoshis) dropped by defeated enemies
3. **Spend sats** at Bitcoin shops (appear every 3 waves)
4. **Convert** enemies with Wake Up (Q) — 2x bonus sats + recruit allies
5. **Survive** boss fights every 5 waves
